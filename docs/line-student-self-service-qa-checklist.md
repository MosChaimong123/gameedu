# LINE Student Self-Service QA Checklist

ใช้ checklist นี้ทดสอบ Phase 3: นักเรียนเชื่อม LINE, ใช้คำสั่งส่วนตัว และครูตรวจสถานะ/รีเซ็ตได้จากหน้าห้องเรียน

## Preconditions

- Render/Local มี `LINE_CHANNEL_SECRET`, `LINE_CHANNEL_ACCESS_TOKEN`, `LINE_CLASSROOM_BINDING_SECRET`
- LINE webhook ชี้ไป `/api/webhooks/line`
- บอทถูกเพิ่มเป็นเพื่อนอย่างน้อย 1 บัญชี
- บอทถูกเชิญเข้ากลุ่ม LINE ของห้องเรียนทดสอบ
- ห้องเรียนใน GameEdu มีนักเรียนอย่างน้อย 2 คน และมีงานอย่างน้อย 1 งาน

## Teacher Visibility

- [ ] เปิดหน้าห้องเรียนใน dashboard
- [ ] เปิดแผง `LINE / งานมอบหมาย`
- [ ] เห็นจำนวน `เชื่อมแล้ว`, `ยังไม่เชื่อม`, และ `พร้อมใช้ %`
- [ ] เห็นรายชื่อนักเรียนทุกคนพร้อม badge `เชื่อมแล้ว` หรือ `รอเชื่อม`
- [ ] นักเรียนที่เชื่อมแล้วแสดงเวลาที่เชื่อมและ LINE id แบบย่อ
- [ ] นักเรียนที่ยังไม่เชื่อมแสดงคำแนะนำให้ล็อกอินแล้วกด `เชื่อม LINE`

## Student Linking

- [ ] นักเรียนล็อกอินหน้าเว็บด้วยบัญชี/รหัสของตน
- [ ] กดปุ่ม `เชื่อม LINE`
- [ ] คัดลอกคำสั่ง `เชื่อม <code>`
- [ ] เปิดแชตส่วนตัวกับ LINE bot
- [ ] ส่งคำสั่ง `เชื่อม <code>`
- [ ] บอทตอบว่าสำเร็จ พร้อมชื่อห้องและชื่อนักเรียน
- [ ] Dialog ในเว็บอัปเดตเป็นสถานะเชื่อมแล้ว
- [ ] หน้าครูรีเฟรชแล้วเห็นนักเรียนคนนั้นเป็น `เชื่อมแล้ว`

## Private LINE Commands

- [ ] ในแชตส่วนตัวพิมพ์ `งานของฉัน`
- [ ] บอทตอบรายการงานค้างของนักเรียนคนนั้น
- [ ] ในแชตส่วนตัวพิมพ์ `คะแนนของฉัน`
- [ ] บอทตอบคะแนนเฉพาะของนักเรียนคนนั้น
- [ ] ในแชตส่วนตัวพิมพ์ `ส่งอะไรแล้ว`
- [ ] บอทตอบรายการงานที่ส่งแล้ว
- [ ] ถ้านักเรียนยังไม่เชื่อม LINE แล้วพิมพ์คำสั่งส่วนตัว บอทต้องให้ไปเชื่อมก่อน

## Group Privacy

- [ ] ในกลุ่ม LINE พิมพ์ `งานค้าง`
- [ ] บอทตอบภาพรวมของห้อง ไม่แสดงรายชื่อนักเรียนรายคน
- [ ] ในกลุ่ม LINE นักเรียนที่เชื่อมแล้วพิมพ์ `งานของฉัน`
- [ ] ในกลุ่ม LINE ต้องเห็นเพียงข้อความ acknowledgement ว่าส่งรายละเอียดให้แชตส่วนตัวแล้ว
- [ ] รายละเอียดงานส่วนตัวต้องไปปรากฏในแชตส่วนตัวเท่านั้น
- [ ] ทำซ้ำกับ `คะแนนของฉัน` และ `ส่งอะไรแล้ว`

## Teacher Reset

- [ ] ในแผงครูกด `รีเซ็ต LINE` ของนักเรียนที่เชื่อมแล้ว
- [ ] ยืนยัน dialog ของ browser
- [ ] ระบบแสดง toast ว่ารีเซ็ตสำเร็จ
- [ ] นักเรียนคนนั้นกลับเป็นสถานะ `รอเชื่อม`
- [ ] นักเรียนพิมพ์ `งานของฉัน` ในแชตส่วนตัวแล้วควรถูกขอให้เชื่อมใหม่
- [ ] นักเรียนสามารถกด `เชื่อม LINE` บนเว็บและเชื่อมใหม่ได้สำเร็จ

## Regression

- [ ] `Send LINE` งานรายชิ้นยังส่งเข้ากลุ่มได้
- [ ] `ทวงงานค้างทั้งห้อง` ยังส่งเข้ากลุ่มได้
- [ ] `Export` งานยังดาวน์โหลดได้
- [ ] นักเรียนที่ไม่เกี่ยวข้องไม่เห็นข้อมูลส่วนตัวของเพื่อนในกลุ่ม
